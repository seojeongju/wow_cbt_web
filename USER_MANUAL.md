# WOW3D 판매 자재관리 프로그램 사용자 매뉴얼

## 1. 프로그램 개요
**WOW3D 판매 자재관리 프로그램**은 주얼리, 소형 카메라, 전자제품 등 재고 관리가 중요한 소규모 비즈니스를 위한 전문 ERP 솔루션입니다. 직관적인 UI와 강력한 시리얼 넘버 추적 기능을 통해 판매와 자재를 효율적으로 통합 관리할 수 있습니다.

### 주요 특징
- **멀티테넌트 SaaS**: 별도의 설치 없이 웹 브라우저만으로 즉시 사용 가능
- **정밀한 재고 관리**: 상품 옵션(Variant)별 재고 및 시리얼 넘버 개별 추적 지원
- **통합 판매 관리**: 주문 접수부터 출고, 배송까지 원스톱 처리
- **스마트한 대시보드**: 실시간 재고 현황 및 판매 트렌드를 한눈에 파악

## 2. 시작하기

### 2.1 로그인
프로그램 접속 주소: [https://wow3d-stock-sales-manager.pages.dev/login](https://wow3d-stock-sales-manager.pages.dev/login)

1. 웹 브라우저에서 접속 주소로 이동합니다.
2. 할당받은 **이메일**과 **비밀번호**를 입력합니다.
3. **[로그인]** 버튼을 클릭하여 대시보드로 이동합니다.

## 3. 주요 기능 및 사용법

### 3.1 대시보드 (Dashboard)
시스템 접속 시 가장 먼저 표시되는 화면으로, 비즈니스의 핵심 지표를 요약하여 제공합니다.

- **주요 지표 (KPI)**: 총 매출액, 주문 건수, 재고 경고 알림 등 핵심 현황 카드 표시
- **최근 주문 내역**: 실시간으로 들어온 최신 주문을 리스트업
- **판매 트렌드**: 기간별 매출 추이를 그래프로 시각화
- **재고 알림**: 재고가 부족한 품목(Low Stock)을 우선적으로 경고

### 3.2 상품 관리 (Product Management)
판매할 상품을 등록하고 관리하는 메뉴입니다. 상품의 기본 정보와 구체적인 옵션(색상, 사이즈 등)을 설정합니다.

- **상품 등록**:
    - 상품명, 카테고리, 설명, 기본 가격 입력
    - 상품 이미지 업로드 및 관리
- **상품 옵션(Variant) 설정**:
    - 동일 상품 내 색상, 사이즈, 용량 등 옵션별 구분 등록
    - 옵션별 별도 SKU(재고 관리 코드) 부여
    - 옵션별 추가 가격 설정 가능

### 3.3 재고 관리 (Inventory Management)
등록된 상품 옵션별로 정확한 수량을 파악하고 관리합니다.

- **재고 현황 조회**:
    - 상품/옵션별 현재고 수량 실시간 확인
    - 상태 필터링: `재고 있음`, `재고 부족(10개 이하)`, `품절`
- **시리얼 넘버 추적**:
    - 고가 제품이나 보증이 필요한 제품의 경우 `시리얼 추적` 기능 사용
    - 개별 상품의 고유 번호(S/N), 입고일, 보증 기간 등 상세 이력 관리
- **입/출고 관리**:
    - 수동 재고 조정 및 입출고 사유 기록

### 3.4 주문 관리 (Order Management)
고객으로부터 발생한 주문을 처리하고 상태를 관리합니다.

- **주문 생성**:
    - 고객 선택 -> 상품 및 수량 선택 -> 주문 생성
    - 주문 생성 시 자동으로 재고 수량 차감 (선입선출 혹은 지정 출고)
- **주문 상태 변경**:
    - `주문 대기` -> `처리중` -> `배송중` -> `완료` 단계별 상태 업데이트
    - 주문 취소 시 재고 자동 복구 기능
- **주문 내역 조회**: 날짜별, 고객별, 상태별 주문 검색

### 3.5 고객 관리 (Customer Management)
B2B 거래처 및 일반 고객의 정보를 관리합니다.

- **고객 등록**: 상호명, 담당자, 연락처, 배송지 주소 등 정보 입력
- **거래 이력**: 특정 고객의 과거 주문 내역 및 미수금 현황 파악
- **고객 통계**: 우수 고객 및 자주 구매하는 상품 분석

### 3.6 설정 (Settings)
시스템의 기본적인 환경을 설정합니다.

- **회사 정보**: 사업자 정보 로고, 연락처 수정 (거래명세서 출력 시 반영)
- **사용자 관리**: 직원 계정 추가 및 권한(관리자/일반) 부여
- **알림 설정**: 재고 부족 알림 기준 수량 등 설정

## 4. 자주 묻는 질문 (FAQ)

**Q. 상품 옵션별로 재고를 따로 관리할 수 있나요?**
A. 네, 가능합니다. [상품 관리]에서 색상, 사이즈 등 옵션을 생성하면, [재고 관리] 메뉴에서 각 옵션별(Variant)로 수량이 독립적으로 카운트됩니다.

**Q. 시리얼 넘버가 있는 제품은 어떻게 관리하나요?**
A. 상품 등록 시 '시리얼 추적' 기능을 켜시면, 재고 수량뿐만 아니라 개별 제품의 고유 시리얼 넘버를 시스템에 등록하여, 판매 시 어떤 시리얼 번호가 출고되었는지 추적할 수 있습니다.

**Q. 재고가 부족하면 알림이 오나요?**
A. 네, 재고 수량이 설정된 기준(기본 10개) 이하로 떨어지면 대시보드와 재고 목록에 '재고 부족(Low Stock)' 경고가 표시됩니다.

## 부록: 프로그램 소개 웹사이트 구축을 위한 핵심 요약

안티그래비티 바이브 코딩 프로그램을 통한 웹사이트 구축 시 참고할 내용입니다.

### 1. 서비스 가치 제안 (Value Proposition)
- **대상**: 소상공인 (소매업, 식음료, 주얼리 등)
- **핵심 메시지**: "복잡한 재고관리, 월 2만원대로 스마트하게 자동화하세요."
- **정부 지원 사업 적합성**:
  - **SaaS 구독형**: 초기 도입 비용 부담 없는 월정액 서비스 (Basic 2.5만원/월 예상)
  - **데이터 분석**: 단순 장부 기록을 넘어선 매출 추이 및 재고 회전율 분석 제공

### 2. 주요 기능 하이라이트 (웹사이트 섹션용)
1. **실시간 대시보드**: 매장의 오늘 매출과 재고 현황을 한 눈에 확인.
2. **시리얼 넘버 추적**: 고가 상품의 개별 이력을 관리하여 분실 및 AS 분쟁 예방.
3. **간편한 주문 처리**: 클릭 몇 번으로 주문 접수부터 재고 차감까지 자동화.
4. **모바일 최적화**: 매장 밖에서도 스마트폰으로 실시간 확인 가능.
